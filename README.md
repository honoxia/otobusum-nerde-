# Otobüsüm Nerde

## Kurulum

1) Bağımlılıkları yükleyin:

```bash
npm install
```

2) Ortam değişkenlerini ayarlayın:

```bash
cp .env.example .env
```

`.env` dosyasına Google Maps API key ekleyin:

```
EXPO_PUBLIC_GOOGLE_MAPS_API_KEY=your_google_maps_api_key_here
```

3) Uygulamayı başlatın:

```bash
npx expo start
```
